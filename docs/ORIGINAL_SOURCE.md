## btFEM - Original Visual Basic Implementation

**Source**: CodeProject article "Finite Element Analysis in VB.NET"  
**Archive**: `btFEM-original-VB-source.zip`  
**Original Author**: [CodeProject Community]  
**Date**: 2018

### About

The `btFEM-original-VB-source.zip` archive contains the original Visual Basic .NET implementation of the finite element analysis solver, which served as the foundation for the modernized C# port (FEMaster 2D).

This archive is provided for reference and historical documentation purposes.
